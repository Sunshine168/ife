# 网页抓取分析服务系列之一（基础分析）
使用的是[phantom](https://github.com/amir20/phantomjs-node)

使用前需要

~~~
npm install
~~~

运行要求

需要在node 7.x以上

使用指令:

~~~
node --harmony-async-await demo1.js keyword
//keyword 是需要搜索的关键词
~~~

附带指令

~~~
node --harmony-async-await demo1.js keyword pic
//keyword 是需要搜索的关键词
~~~

获得当前页面的图片并渲染出来
